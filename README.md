# ss_task4

## gcd.sh

### 用途: 
		最大公約数を出力する
      
### 書式: 
		gcd.sh num1 num2
     
### 解説: 
		2つの自然数を引数とする
		1つの自然数の最大公約数を出力とする
		その他、正しくない入力を行った際はエラー終了する
       
### 戻り値:
		0   正常終了
		1   ユーザの入力間違い

## test.sh

### 解説
		最大公約数シェルスクリプトに様々な入力を行う
		想定した挙動をしていない場合にエラーを表示して
		終了する。エラーがなければ、なにも出力しない。
